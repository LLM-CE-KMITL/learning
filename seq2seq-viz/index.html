SEQ2SEQ POLYLINE LAB

A tiny vanilla encoder–decoder RNN, trained live in your browser as a proper seq2seq model: the encoder reads a whole input sequence, and both a plain decoder and a self-attention decoder (sharing the same encoder) learn to regenerate it as a sequence of absolute [x, y] points via teacher forcing - the attention decoder's query evolves at every output step, so it can attend to different input positions as it generates.
