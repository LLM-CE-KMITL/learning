## About

Welcome to the **LLM Learning Hub** — an interactive educational initiative designed to make machine learning, deep learning, and language models **visual and intuitive**.

Developed by the **SAIG Lab** at **King Mongkut's Institute of Technology Ladkrabang (KMITL)**, these visualizers break down complex AI architectures into real-time, interactive demonstrations — so you can *see* what's happening under the hood, not just read about it.

---

## Machine Learning

- [**Linear Regression**](https://llm-ce-kmitl.github.io/learning/ml-viz/linear.html) — Watch a model fit a line to data in real time
- [**Multilayer Perceptron**](https://llm-ce-kmitl.github.io/learning/ml-viz/mlp.html) — Explore how neurons and layers combine to learn
- [**Activation Functions**](https://llm-ce-kmitl.github.io/learning/ml-viz/activations.html) — Visualize ReLU, Sigmoid, Tanh and more

## Deep Learning

- [**RNN**](https://llm-ce-kmitl.github.io/learning/dl-viz/rnn.html) — See how recurrent networks process sequences
- [**Seq2Seq**](https://llm-ce-kmitl.github.io/learning/seq2seq-viz/rnn.html) — Explore encoder-decoder sequence modeling

## Language Models

- [**Word2Vec (CBOW)**](https://llm-ce-kmitl.github.io/learning/word2vec-viz/cbow.html) — Understand how words become vectors
- [**Transformer (Text)**](https://llm-ce-kmitl.github.io/learning/transformer-viz/text-gen.html) — Step through text generation, token by token
- [**Transformer (Line)**](https://llm-ce-kmitl.github.io/learning/transformer-viz/line-gen.html) — Visualize transformer attention on sequences
